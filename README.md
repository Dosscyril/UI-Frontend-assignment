# PopX UI Frontend Assignment

This project is a **pixel-perfect implementation of the PopX mobile UI design** built using React.
The interface replicates the provided design and includes multiple pages with smooth navigation.

---

## Tech Stack

* **React JS** – Frontend framework
* **Tailwind CSS** – Styling
* **React Router DOM** – Page navigation
* **Create React App** – Project setup

---

## Features

* Pixel-perfect UI implementation based on the design
* Mobile UI centered on the webpage
* Multiple pages with smooth navigation
* Reusable React components
* Clean project structure

---

## Pages Implemented

1. Welcome Page
2. Create Account Page
3. Login Page
4. Account Settings Page

## Installation & Setup

Clone the repository:

```
git clone https://github.com/Dosscyril/UI-Frontend-assignment.git
```

Navigate to the project folder:

```
cd UI-Frontend-assignment
```

Install dependencies:

```
npm install
```

Run the project:

```
npm start
```
